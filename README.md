# LeftWM themes - windowsxp

If you want to use them copy them into:
~/.config/leftwm/themes

symlink the theme you want to try to current

```bash
cd ~/.config/leftwm/themes
cp THEME_FROM_THIS_REPO .
ln -s THEME_NAME current
```
